Bowling Score Calculator

This is a Ten-pin bowling score calculator written in Haskell.

Note that the name of the asked for function is not CalculateScore but calculateScore because of the conventions of Haskell.

- Written on Arch Linux
- Type "make" in the command line and run a.out with "./a.out "argumnets" "
- Example:
    ./a.out "53 72 x x 7/ 9/ 24 6/ x x x 5"
